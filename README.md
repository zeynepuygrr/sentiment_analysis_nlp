# 🎬 Sentiment Analysis Application

## Project Overview
A text-based sentiment analysis application built for the NLP course project.
The system classifies movie reviews as **Positive** or **Negative** using two different models.

## Models Used
| Model | Accuracy | F1-Score |
|-------|----------|----------|
| TF-IDF + Neural Network (PyTorch) | 84% | 0.81 |
| DistilBERT Transformer (Fine-tuned) | 88% | 0.88 |

## Technologies
- Python, PyTorch, HuggingFace Transformers
- TF-IDF (Scikit-learn), Gradio UI
- Dataset: IMDB Movie Reviews (50,000 reviews)

## How to Run
1. Open the notebook in Google Colab
2. Run all cells in order
3. Use the Gradio interface to test the model
